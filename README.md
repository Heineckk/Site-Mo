# Site Ana Lívia

Site romântico 3D feito com Next.js, Three.js, Framer Motion e GSAP.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000)

## Publicar no GitHub

1. Instale o [GitHub CLI](https://cli.github.com/) (se ainda não tiver)
2. Faça login:

```bash
gh auth login
```

3. Envie o projeto:

```powershell
.\push-github.ps1
```

## Colocar online (Vercel)

1. Acesse [vercel.com](https://vercel.com) e entre com GitHub
2. **Add New → Project** → selecione o repositório `ana-livia`
3. Clique em **Deploy**
4. Envie o link gerado para ela
