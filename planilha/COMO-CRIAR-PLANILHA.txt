CONTROLE DE PEDIDOS — DOTTI 303 (VERSÃO PROFISSIONAL)
======================================================

COMO CRIAR
----------
1. Abra: https://script.google.com
2. Novo projeto → cole o código de "criar-planilha.gs" → Salvar
3. Selecione a função: criarPlanilhaControlePedidos
4. Clique em Executar ▶ (NÃO clique em Implantar)
5. Autorize o Google
6. Abra o link que aparecer no Registro de execução


O QUE VEM NA PLANILHA
---------------------

📊 Dashboard (aba principal)
   • Cards com totais: pedidos, aguardando, em produção, entregues
   • Faturamento e ticket médio
   • Gráficos automáticos por status e por modelo

📋 Pedidos
   • Cabeçalho com visual Dotti 303 (roxo/rosa)
   • Menus dropdown em Status, Modelo, Pago?, etc.
   • Cores automáticas por status
   • Filtros no cabeçalho para buscar pedidos
   • Linhas zebradas para leitura fácil

📖 Como usar
   • Instruções rápidas dentro da planilha


DICA
----
Preencha cada novo pedido na aba «Pedidos», linha 4 em diante.
O Dashboard atualiza sozinho.


DASHBOARD MOSTRA 0?
-------------------
1. Confira se os pedidos estão na aba «Pedidos» (não só no CSV).
2. Preencha pelo menos a coluna «Cliente» — o total usa ela, não a Data.
3. Status deve ser exatamente do menu: Aguardando pagamento, Pago, Em produção, Entregue ou Cancelado.
4. Para corrigir uma planilha já criada:
   - Abra a planilha no Google Sheets
   - Extensões → Apps Script
   - Cole o código atualizado de criar-planilha.gs
   - Execute a função: atualizarDashboard
   - Volte na aba Dashboard — os números devem aparecer.
